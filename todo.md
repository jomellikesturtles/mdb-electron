Title: Download App Button

Description:

Display download button if the user is accessing app by browser

Acceptance Criteria:

- URL of download is: https://github.com/jomellikesturtles/mdb-electron/releases/download/v1.0.0-alpha/mdb-darwin-arm64.zip
- Do not show download if they are already using Electron/Desktop app
- MAC OS: Shwo button
- Windows/Linux: Show the button but disabled; add tooltip that app is only available for MAC OS
- Mobile: Show the button; show modal message that app is only available for MAC OS
